# 店舗用アートメイク電子同意書アプリ

店側のスマホ、iPad、タブレットで開き、お客様に入力、確認、電子署名してもらうためのWebアプリです。お客様側でアプリをダウンロードする必要はありません。

## 画面

- お客様入力画面: `/index.html`
- 管理画面: `/admin.html`

## 管理画面

初回は `/index.html` でサロン登録を行います。登録したメールアドレスとパスワードで、入力画面と管理画面にログインできます。

管理画面では、同意書一覧、氏名検索、電話番号検索、施術日検索、施術部位検索、PDF表示、PDFダウンロード、印刷、削除ができます。

## Firebase連携

`firebase-config.js` の `enabled` を `true` にし、Firebaseプロジェクトの接続情報を入力してください。`salonId` ごとに `salons/{salonId}/consents` へ保存されるため、サロンごとのデータ分離ができます。

Firebase Authenticationではメールアドレス/パスワードログインを有効にし、サロン用スタッフアカウントを作成してください。Firebase未設定の間は、これまで通りブラウザのローカル保存で動作します。

## Netlifyへのアップロード

1. このフォルダ内のZIPファイルをNetlifyへドラッグ&ドロップします。
2. 発行されたURLを店側端末で開きます。
3. iPhone/iPadの場合はSafariの共有メニューから「ホーム画面に追加」を選びます。
4. Androidの場合はChromeのメニューから「ホーム画面に追加」を選びます。

## 運用方法

1. 店側端末でアプリを開きます。
2. 初回のみサロン名、メールアドレス、パスワードを登録します。
3. ログイン後、ホーム画面から「電子同意書」を選びます。
4. 端末をお客様に渡し、入力、チェック、電子署名をしてもらいます。
5. 送信後、入力フォームと署名は自動でクリアされます。
6. 「次のお客様の入力へ」から次のお客様がすぐ使えます。

## 注意

現在の保存先はブラウザのローカル保存です。同じ端末・同じブラウザ内で管理画面から確認できます。複数端末で共有する場合は、FirebaseやGoogleスプレッドシート連携への拡張を行ってください。
